# ニュースデータの管理方法

## 概要

ニュースは`news.json`ファイルで管理されています。HTMLファイルを直接編集する必要はありません。

## ニュースの追加・更新方法

### 1. `news.json`ファイルを編集

`assets/data/news.json`ファイルを開き、以下の形式でニュースを追加または編集してください。

```json
{
  "date": "2024.12.15",
  "title": {
    "ja": "ニュースのタイトル（日本語）",
    "en": "News Title (English)"
  },
  "content": {
    "ja": "ニュースの本文（日本語）",
    "en": "News content (English)"
  }
}
```

### 2. 日付形式

- 形式: `YYYY.MM.DD`（例: `2024.12.15`）
- 日が不明な場合: `YYYY.MM.00`（例: `2024.12.00`）
- 月が不明な場合: `YYYY.00.00`（例: `2024.00.00`）

### 3. 配列の順序

配列の順序は関係ありません。JavaScriptが自動的に日付順（新しい順）にソートして表示します。

### 4. 例

新しいニュースを追加する場合：

```json
[
  {
    "date": "2024.12.15",
    "title": {
      "ja": "新規研究プロジェクト開始",
      "en": "New Research Project Launched"
    },
    "content": {
      "ja": "新しい研究プロジェクトが開始されました。",
      "en": "A new research project has been launched."
    }
  },
  // ... 既存のニュース ...
]
```

## 注意事項

- JSONファイルの構文エラーに注意してください（カンマ、引用符など）
- 日本語と英語の両方のタイトルと本文を必ず記入してください
- ファイルを保存後、ブラウザでページをリロードすると変更が反映されます
